# SentinelOps Repo Guidance

SentinelOps is a plug-and-play incident and operations copilot.

Default priorities in this repo:

1. Keep CLI and repo-local install behavior correct.
2. Keep generated agent/editor integrations correct.
3. Keep `.sentinelops/project.toml` as the repo-local single-config source of truth.
4. Keep README, validation docs, and production/commercial guidance aligned with the code.
