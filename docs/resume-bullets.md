# Resume Bullets

## Short version

- Built `SentinelOps`, a local-first incident copilot that analyzes logs, retrieves supporting runbooks and notes, and executes approval-aware investigation workflows using FastAPI, Ollama, Chroma, and LangGraph.
- Designed a production-minded API layer with readiness checks, structured problem-detail errors, runtime metrics, caching, retries, Docker packaging, and CI-backed smoke coverage.
- Added an operator console, incident library, saved incident timeline, deterministic evaluation reporting, architecture documentation, and communication assets for operational walkthroughs.

## Stronger technical version

- Engineered a laptop-friendly AIOps application that combines structured incident analysis, safe local tool use, semantic retrieval, and checkpointed workflows without depending on cloud infrastructure.
- Implemented a durable workflow service with SQLite checkpoints, approval pauses, audit history, and thread inspection to support reliable remediation planning instead of brittle one-shot summaries.
- Added observability and resilience features including request tracing hooks, runtime metrics, bounded caches, dependency readiness reporting, and graceful degradation when Ollama or retrieval features are unavailable.
- Built an operator-facing console and incident library that make realistic incident walkthroughs reproducible while validating behavior against deterministic local evaluations.
